# CppRTOS

Modern Real-Time Operating System written in C++ for ARM Cortex-M microcontrollers

CppRTOS is a lightweight, preemptive real-time operating system designed specifically for ARM Cortex-M microcontrollers. The project focuses on deterministic execution, modern C++ design, portability, and ease of integration into embedded applications.

Unlike traditional C-based RTOS implementations, CppRTOS provides an object-oriented API while maintaining predictable real-time behavior and low runtime overhead.

Project Status: Preview Release (Core functionality implemented, API still evolving)
